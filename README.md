## Qué es este proyecto

Construcción del nuevo sitio web institucional de Alianz Canine Worldwide, organización cinológica internacional con presencia en 50+ países. Reemplaza dos sitios existentes (`alianzfederation.org` y `alianzcanineworldwide.org`) con una sola presencia profesional en GHL.
